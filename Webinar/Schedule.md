# Azure AD webinar にご参加ください <br>(セッション情報随時追加、Bookmark 推奨!)

- Office 365 管理者のみなさま
- Active Directory 管理者の皆様
- 組織の ID 管理基盤の計画、導入、保守担当者のみなさま
- 組織のセキュリティ施策の計画、導入、保守担当者のみなさま
